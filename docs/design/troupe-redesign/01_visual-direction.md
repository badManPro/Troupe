# Troupe Visual Direction

## Goal

Restyle the existing Troupe app to match the provided PNG references.  
The app already has full functionality. Do not rewrite business logic, routing, data loading, or state management unless absolutely necessary.  
Focus on visual restyling, component polish, layout hierarchy, theming, and responsive behavior.

## Core Mood

New-generation AI-native workspace.  
Dark-first atmosphere.  
Light glassmorphism, delicate glow, quiet AI-thinking feeling.  
Somewhere between v0.dev, Cursor, and Anthropic, but not cyberpunk and not flashy.

## Color

Dark base:
- #0B0D12
- #111318

Accent colors:
- Soft violet: #A78BFA
- Mint: #6EE7B7
- Warm orange: #FB923C

Light mode:
- frosted glass white, not pure white
- low contrast neutral surfaces
- subtle translucent borders

## Typography

- English: Geist Sans
- Chinese: HarmonyOS Sans
- Code: JetBrains Mono
- Display title: Söhne-like display feeling, used sparingly

## Shape

- Large cards: 12–18px radius
- Buttons: medium radius
- Badges: pill
- Cards: 1px translucent border + subtle inner glow

## Motion

Use subtle motion:
- soft fade-in for streaming text
- relay-like progress animation
- subtle parallax
- soft glow movement

Implementation should be restrained. Motion must not distract.

## Decoration

No traditional illustration.  
Use abstract geometry, grid light spots, soft particles, and glowing line icons.
